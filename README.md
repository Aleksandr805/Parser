# Parser
<div>
  <h2>📦 Краткая инструкция: Парсер цен для Excel</h2>

  <section>
    <h3>🎯 Назначение</h3>
    <p>Автоматически собирает цены товаров с разных интернет‑магазинов по ссылкам из Excel‑файла и сохраняет их в ту же таблицу.</p>
  </section>

  <section>
    <h3>⚙️ Как работает</h3>
    <ol>
      <li>Подготовьте Excel‑файл с колонкой <code>link</code> (ссылки на карточки товаров).</li>
      <li>Откройте программу, выберите файл и нажмите «Запустить парсинг».</li>
      <li>Скрипт автоматически определяет сайт по домену из ссылки, подбирает соответствующий CSS/XPath‑селектор из <code>selectors.json</code> и извлекает цену.</li>
      <li>Результат записывается в колонку <code>price</code>. Строки, где цена уже есть, пропускаются → можно безопасно перезапускать или останавливать процесс.</li>
    </ol>
  </section>

  <section>
    <h3>📄 Требования к файлу</h3>
    <p>Достаточно одной колонки с именем <code>link</code>. Остальные столбцы (бренд, модель, артикул и т. д.) скрипт не трогает.</p>
  </section>

  <section>
    <h3>✨ Ключевые возможности</h3>
    <ul>
      <li>🌐 Автоопределение сайта по домену (не нужно создавать отдельные колонки <code>link_сайт</code>)</li>
      <li>🔄 Два метода парсинга: быстрый <code>Requests</code> и headless‑браузер <code>Selenium</code> (режим <code>Auto</code> пробует оба)</li>
      <li>🛡️ Безопасное сохранение: запись во временный файл → замена оригинала (защита от битых Excel)</li>
      <li>⏸️ Кнопка «Остановить» без потери уже собранных данных</li>
      <li>🖥️ Встроенный редактор селекторов с тестированием прямо из GUI</li>
    </ul>
  </section>

  <section>
    <h3>📌 Важно</h3>
    <ul>
      <li>Селекторы и настройки сайтов хранятся в <code>selectors.json</code> (вкладка «Обучение»).</li>
      <li>Пути к <code>ChromeDriver</code>, таймауты и тема интерфейса настраиваются во вкладке «Настройки».</li>
      <li>Все ошибки, прогресс и логи выводятся в окне консоли в реальном времени.</li>
    </ul>
  </section>
</div>
