# Parser
Универсальный парсер цен по ссылкам, с обучением, по селекторам.
Краткая инструкция: Парсер цен для Excel
🎯 Назначение
Автоматически собирает цены товаров с разных интернет-магазинов по ссылкам из Excel-файла и сохраняет их в ту же таблицу.
Как работает
Подготовьте Excel-файл с колонкой link (ссылки на карточки товаров).
Откройте программу, выберите файл и нажмите «Запустить парсинг».
Скрипт автоматически определяет сайт по домену из ссылки, подбирает соответствующий CSS/XPath-селектор из selectors.json и извлекает цену.
Результат записывается в колонку price. Строки, где цена уже есть, пропускаются → можно безопасно перезапускать или останавливать процесс.
Требования к файлу
Достаточно одной колонки с именем link. Остальные столбцы (бренд, модель, артикул и т.д.) скрипт не трогает.
Ключевые возможности
• Автоопределение сайта по домену (не нужно создавать отдельные колонки link_сайт)
• Два метода парсинга: быстрый Requests и headless-браузер Selenium (режим Auto пробует оба)
• Безопасное сохранение: запись во временный файл → замена оригинала (защита от битых Excel)
• Кнопка «Остановить» без потери уже собранных данных
• Встроенный редактор селекторов с тестированием прямо из GUI
Важно
• Селекторы и настройки сайтов хранятся в selectors.json (вкладка «Обучение»).
• Пути к ChromeDriver, таймауты и тема интерфейса настраиваются во вкладке «Настройки».
• Все ошибки, прогресс и логи выводятся в окне консоли в реальном времени.
